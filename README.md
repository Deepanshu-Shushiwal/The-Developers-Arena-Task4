# E-Commerce Sales Analysis

## Description
This project performs a complete data analysis workflow on e-commerce sales data using Python.

## Tools Used
- Python
- Pandas
- Matplotlib

## Features
- Data cleaning and validation
- Sales analysis
- Multiple visualizations
- Professional reporting

## How to Run
1. Place sales_data.csv inside data folder
2. Run analysis.py
3. View generated charts inside charts folder
